CyberLearnHub Forensics Challenge - Locked Archive
====================================================

You found an encrypted backup archive: secure_backup.zip

It is protected with a password. A previous admin left themselves
a note so they wouldn't forget it - but they "encrypted" their own
note with a simple substitution cipher before leaving the office.

Admin's note (ROT13 encoded):
"gur cnffjbeq vf a3gj0ex99"

Decode the note to recover the password, then extract the archive
to find the flag inside.
